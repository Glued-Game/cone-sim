### Cone-SIM

A simple self-driving car simulator built with Unity.

  - Easy to create new tracks
  - Save training data in txt files
  - Udp interface with autonomous mode

### Training Mode

<p align="center">
<img src="images/pic_2.PNG" width="450" height="270">
</p>

Data on the following table is store which frame

| Index | Data |
| ------ | ------ |
| 0 | Recording time |
| 1 | Throttle input |
| 2 | Brake input |
| 3 | Steering input |
| 4 | Handbrake input |
| 5 | Car speed |
| 6 | Acceleration x |
| 7 | Acceleration z |
| 8 | Position x |
| 9 | Position z |
|10 to 45 | 36 raycasts distance |
